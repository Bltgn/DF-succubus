Les pouvoirs

Noms : Temptation, villany, vomluptuous, sassy (saucy)

! tester
+ reagents
> plus tard

Roles
- Effets sur les invaders
- Effets sur le moral
- Effet sur l'exterieur
- Upgrades pour le combat
- Upgrades civils
- Sorts offensifs

Aide
- Créer un plugin pour afficher les détails d'une réaction.
- Faire en sorte que l'on choisisse des workers dédiés
- Ajouter des indicateurs style masterwork

= 7 temples =

Lust
- Upgrades + effets grace aux pets
- Reagents : Instruments, toys
- Multiplicateur : Happiness
- Modif : Altruism, autres.
- Contrecout : Le worker ajouter un lover dans les relations.

+ Entice (perte de skill enemy, peut faut tomber les armes)
