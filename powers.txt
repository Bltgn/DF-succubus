Les pouvoirs

Roles
- Effets sur les invaders
- Effets sur le moral
- Effet sur l'exterieur
- Upgrades pour le combat
- Upgrades civils
- Sorts offensifs

Aide
- Utiliser plusieurs inorganics pour décrire des effets, couts etc...
- Faire en sorte que l'on choisisse des workers dédiés
- Ajouter des indicateurs style masterwork
- Créer un plugin pour afficher les déails, tout le temps

= 7 temples =

Lust
- Upgrades offensifs
- Reagents : Extracts, drinks
- Multiplicateur : Happiness
- Modif : Altruism, autres.
- Contrecout : Le worker peut ajouter un lover dans les relations.

* Release pheromones (aoe debuff, buff les autres succubi)
* Climax (up des stats en combat, complètement OP à ecstatic)
* Broken heart (rip heart)

Greed
- Upgrades civils
- Reagents : crafts
- Multiplicateur : Valeur du craft
- Modif : immoderation, craving, short term gain
- Contrecout : ?

* Influence (?)
* Posess a distant worker (+random skill, si un artefact est utilisé get legendary, annonce et set mood à 1)

Sloth
- Améliorateur de productivité
- Reagents : Plantes? gemmes?
- Multiplicateur : Noble (+1 pour les admins, +5 pour la reine)
- Modif : Lazy
- Contrecout : Le worker prend une pause

* Gain d'interaction pour réduire les pauses
* Annuler les pauses

Pride
- Améliorateur de moral
- Reagents : Crafts, furnitures, toys...?
- Multiplicateur : Nombre de relations
- Modifs : Immodesty, immoderation
- Contrecout : Chances de siege par les deep succubi, le worker se fait un ami.

* Forget death (oubli des morts)
* Get summoned (le worker est téléporté, exilé, gros gain de moral)

Gluttonry
- Effets grace aux pets
- Reagent : Nourriture, souls
- Multiplicateur : Qualité de la nourriture employée
- Modifs : Skill de cuisine, fat
- Contrecout : Faim et/ou soif

* Coiling nahashes (up de moral grace aux nahash)
* Entangling tentacles (defense+ à porté des tentacle monsters)

Wrath
- Upgrades pour le combat
- Multiplicateur : Meilleur skill de combat
- Modifs : Colère, prone to rage
- Contrecout : Aucun

* Reinforce wings (devient volant)
* Berzerk rage (en combat + force et no exert)
* Reinforce strengh
* Reinforce stamina
* Reinforce agility
* Slick skin (peau et os plus solide)

Envy
- Effet sur les invaders
- Multiplicateur : Manque de skills (sauf alchemy), doit être non noble.
- Modifs : Liberalism
- Contrecout : Perte de moral pour le worker

* Rendre un invader crazed
* Chef inconscient
* Faire partir des invaders

== Altar royalists ==

Altar of fire
- Blasts offensifs (feu!)
- Upgrades
- Requiert des extract

* Become fireimmune
* Torrid kiss (incinerate au toucher)
* Gehena fire (fireball)
* Infernal rain (exterieur)
* Geyser de feu (1 invader non chef)

Soul forge

- Extraction d'ames de cadavres
- Cristalize soul (gem + soul)

- Construire des lances magiques, dégat faibles, effets en plus (reagent : souls)
* Pain lance : Cause de la douleur
* Force lance : Undirected dust
* Fire lance : Feu

Altorney
- Consomme des objets, gains de souls
